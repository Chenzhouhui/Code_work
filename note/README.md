# note 目录说明

note 文件夹用于 C 语言重要知识点的归纳、数据结构基础与典型代码示例。每个文件聚焦一个知识点或数据结构，适合查阅、复习和实验。

## 文件列表与说明

- **p.c**  指针与结构体综合示例
  - 涵盖指针基础、指针与数组、指针的指针、swap、结构体定义与 typedef、结构体指针、动态分配结构体等。
  - 运行效果：依次输出各知识点的演示结果。

- **list.c**  线性表（顺序表与单链表）
  - 总结线性表的基本概念，分别实现顺序表（静态数组）和单链表（带头结点），并演示插入、删除、查找、遍历等操作。
  - 运行效果：输出顺序表和链表的操作过程与结果。

## 编译与运行方法（Windows, MinGW/gcc）

以 list.c 为例：
```bash
gcc -g list.c -o list.exe
./list.exe
```
其余文件同理，将 list.c 替换为对应文件名。

# git_demo 目录说明

git_demo 文件夹用于 C 语言基础语法与数据结构相关知识点的演示和练习。每个文件聚焦一个典型知识点，适合初学者理解指针、结构体、动态分配等 C 语言核心内容。

## 文件列表与说明

- **1.c**  指针与 swap 函数
  - 演示如何通过指针交换两个变量的值。
  - 运行效果：输出交换前后变量的值。

- **2.c**  结构体与 typedef
  - 定义 `Student` 结构体，演示结构体变量、结构体数组的初始化与遍历。
  - 运行效果：输出每个学生的 id、name、age。

- **3.c**  结构体指针与函数参数
  - 演示如何通过结构体指针参数初始化结构体成员。
  - 运行效果：输出结构体成员赋值后的内容。

- **4.c**  结构体指针与动态分配
  - 演示如何在堆上动态分配结构体，并通过指针访问和释放。
  - 运行效果：输出动态分配结构体的成员内容。

- **5.c**  二级指针（指向指针的指针）
  - 演示如何通过二级指针在函数中为调用者分配内存并赋值。
  - 运行效果：输出分配后指针的地址和值。

## 编译与运行方法（Windows, MinGW/gcc）

以 1.c 为例：
```bash
gcc -g 1.c -o 1.exe
./1.exe
```
其余文件同理，将 1.c 替换为对应文件名。
